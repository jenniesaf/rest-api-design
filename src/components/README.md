# Components

This folder contains shared UI components that can be used across the application.

**Purpose:** Domain-agnostic, reusable UI components (buttons, forms, modals, etc.) that can be used by any feature or app layer.

**Dependency Rules:** This is part of the Shared Layer. It CANNOT import from Features or App layers.

For more details on folder structure and rules, see [README.md](../../README.md).
